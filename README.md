# Scratchpad & Finds

A personal repository for collecting, organizing, and sharing ideas, experiments, research papers, and other interesting finds related to AI and software engineering.

## Purpose

This repo serves as a living notebook — a place to:

- **Curate valuable research papers** with summaries and key takeaways
- **Document experiments and tests** worth revisiting or sharing
- **Capture ideas and insights** before they get lost
- **Share findings** with colleagues and the community

## Repository Structure

```
scratchpad-and-finds/
├── 00_valuable_AI_papers/   # Curated AI research papers with summaries
└── README.md
```

## Contributing

Feel free to add new folders for different topics. Follow the existing naming convention:

- Prefix folders/files with a numeric index (`00_`, `01_`, ...)
- Use snake_case for file and folder names
- For papers: include the original PDF and a markdown summary (`XX_resume_<title>.md`)
